# bi-portal

http://megam.io/

A User Interface for meglytics-BI product.

## Quick Start

1. git clone https://github.com/megamsys/bi-portal.git
2. cd bi-portal
3. npm install
4. node app.js
5. Open <http://localhost:1337>
